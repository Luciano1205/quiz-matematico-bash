# Prompts utilizados con IA

Para el desarrollo de este proyecto se utilizó Claude (Anthropic) como asistente. A continuación, el prompt principal empleado:

## Prompt principal

> Necesito desarrollar un quiz matemático en Bash. Ayúdame con lo siguiente:
> - Generar un ejemplo para implementar números aleatorios en rangos específicos.
> - Genera un ejemplo de un menú interactivo que se repita hasta que el usuario elija salir, validando que la opción ingresada sea correcta y mostrando un mensaje de error si no lo es.
> - Genera el ejemlo de un sistema de 3 intentos donde el usuario pueda volver a responder si falla, salir del bucle al acertar y volver al menú principal.
